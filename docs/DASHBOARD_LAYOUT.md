# Dashboard Analytics - Kerangka & Layout

Document ini menjelaskan semua dashboard yang bisa dibuat dari data yang sudah di-fetch dari Meta Graph API.

---

## 📊 Database Tables & Data Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                        META GRAPH API                            │
│  (pages, ads, catalogs, threads, instagram)                     │
└─────────────────────────┬───────────────────────────────────────┘
                          │
                    ┌─────▼─────┐
                    │meta-fetcher│
                    │  .ts      │
                    └─────┬─────┘
                          │
          ┌───────────────┼───────────────┐
          │               │               │
    ┌─────▼─────┐   ┌─────▼─────┐   ┌─────▼─────┐
    │ Analytics │   │AdPerform. │   │Dashboard  │
    │ table     │   │  table    │   │ Settings  │
    └───────────┘   └───────────┘   └───────────┘
```

### Tables Created:

| Table | Purpose | Data |
|-------|---------|------|
| `analytics` | Daily platform metrics | followers, posts, engagement, reach, views |
| `ad_performance` | Ads campaign metrics | spend, impressions, clicks, CPC, CPM, ROAS |
| `dashboard_settings` | Raw JSON caches | metaAdsData, googleDriveData |
